# Sheethal Joy - Portfolio

Modern, responsive portfolio website built with React + Tailwind CSS + Framer Motion.

## Quick Start

```bash
npm install
npm run dev
```

Visit: http://localhost:5173

## Features

- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Dark/Light mode toggle
- ✅ Smooth animations with Framer Motion
- ✅ Modern gradient UI
- ✅ Contact form
- ✅ Downloadable resume

## Customize

1. **Add your photo**: Replace placeholder in `src/components/Hero.jsx` and `About.jsx`
2. **Update info**: Edit text in each component
3. **Add projects**: Modify `projects` array in `src/components/Projects.jsx`
4. **Social links**: Update links in `src/components/Hero.jsx`
5. **Contact info**: Update in `src/components/Contact.jsx`

## Deploy

### Vercel (Recommended)
```bash
npm install -g vercel
vercel
```

### Netlify
```bash
npm run build
# Drag 'dist' folder to netlify.com
```

## Tech Stack

- React 18
- Tailwind CSS
- Framer Motion
